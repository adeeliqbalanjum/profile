# Adeel Iqbal — Portfolio

Premium static portfolio website for GitHub Pages.

## Files

- `index.html` — site structure and content
- `styles.css` — premium glassmorphism styling, responsive layout, dark/light mode
- `script.js` — smooth interactions, mobile menu, reveal animations, theme toggle
- `.github/workflows/deploy.yml` — GitHub Pages deployment workflow

## GitHub Pages

Go to **Settings → Pages** and set **Source** to **GitHub Actions**. The workflow will deploy the site from the `main` branch.
